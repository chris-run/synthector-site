Privacy Policy

Last updated: February 2026

Data Collection and Processing

This website (synthector.com) is a static informational site hosted on GitHub Pages. We do not collect, store, or process any personal data from visitors.

No Tracking or Analytics

This site does not use:

  • Cookies
  • Analytics services (Google Analytics, etc.)
  • Tracking pixels
  • Third-party advertising services
  • Any form of user tracking or profiling

Server Logs

This site is hosted on GitHub Pages. GitHub may collect technical information in standard server logs, including IP addresses and browser information, as part of their hosting service. This data collection is governed by GitHub's Privacy Statement.

Email Communication

If you contact us via email at Synthector@gmail.com, your email address and the content of your message will be processed solely for the purpose of responding to your inquiry. We do not share this information with third parties or use it for any other purpose.

GDPR Compliance

As this website does not collect or process personal data through the website itself, most GDPR requirements do not apply. However, we respect your privacy rights:

  • Right to be informed: This privacy policy explains our (non-)practices clearly
  • Data minimization: We collect no data through the website
  • Purpose limitation: Email correspondence is used only for responding to inquiries
  • Storage limitation: Email correspondence is retained only as long as necessary

Synthector API Service

If you use the Synthector API service (separate from this website), the following applies: Synthector processes content in-memory and does not retain customer content artifacts.

  • Zero data retention by design: No raw or redacted transcript content is ever stored
  • In-memory processing only: All data is processed in ephemeral memory and immediately purged
  • No persistence layer: The system architecture explicitly prevents data retention
  • Audit metadata only: Only non-content metadata (timestamps, operation types, entity counts) may be logged for system integrity

Third-Party Services

This website is hosted on GitHub Pages and may be served through GitHub infrastructure. The site also loads fonts from Google Fonts (IBM Plex Sans / IBM Plex Mono) to render typography. Other than that, it does not embed analytics, tracking pixels, advertising scripts, or third-party widgets intended to collect user data.

Changes to This Policy

Any changes to this privacy policy will be reflected on this page with an updated "Last updated" date.
